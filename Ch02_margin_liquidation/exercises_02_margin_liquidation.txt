题目 2（保证金 + 爆仓价，含交叉盘与加密货币） / Exercise Set 2 (Margin + Liquidation Price, incl. Cross Pairs and Crypto)

1)
已知：账户杠杆为 1:100，账户净值为 6,000 美元，当前 GOOGL（Alphabet Inc.）价格为 133.45 美元。若客户买入 2.5 手。
Given: leverage 1:100, equity USD 6,000, GOOGL (Alphabet Inc.) at USD 133.45. The client buys 2.5 lots.
— GOOGL 的合约大小是多少？ / Contract size of GOOGL?
ans: 美股CFD的合约量是 100 股/手 / US-stock CFD contract size = 100 shares per lot.

— 点值是多少？ / Pip value?
ans: Pip value = TickSize × Contract = 0.01 × 100 = 1 USD / point

— 所需保证金是多少？ / Required margin?
ans: Margin = Lots × Contract × Price / Leverage = 2.5 × 100 × 133.45 / 100 = 333.63 USD

— 若止损比例为 50%，请问爆仓时的 GOOGL 价格是多少？ / With stop-out ratio 50 %, what is the liquidation price?
ans: Long → price must fall to liquidate. Condition: Equity = Margin × Stop-out Ratio.
     i.e. 6,000 − (133.45 − LiqPrice) × 2.5 × 100 = 333.63 × 0.50
     → Stop-out price = 110.12 USD.

Given: leverage 1:300, equity USD 5,000, HK50 index at 22,959.60, USDHKD 7.64523. The client buys 1.5 lots.
— 合约大小是多少？ / Contract size?
ans: HK50 的合约量是 1 / HK50 contract size = 1.

— 点值是多少？ / Pip value?
ans: Pip value = TickSize × Contract = 0.01 × 1 = 0.01 HKD / point
     Convert to USD: 0.01 / 7.64523 ≈ 0.0013 USD / point.

— 所需保证金是多少？ / Required margin?
ans: Margin in HKD first, then convert to USD:
     Margin(HKD) = Lots × Contract × Price / Leverage = 1.5 × 1 × 22,959.60 / 300 = 114.80 HKD
     Margin(USD) = 114.80 / 7.64523 = 15.01 USD

— 若止损比例为 30%，请问爆仓时的 HK50 价格是多少？ / Liquidation price with 30 % stop-out?
     Margin = 114.80
     5,000 − (22,959.60 − P) × 1.5 = 114.80 × 0.30
     → Stop-out price = 19,649.23.


4)
已知：账户杠杆为 1:500，账户净值为 2,000 美元，EURGBP 报价为：卖出价 0.87230，买入价 0.87260。若客户卖出 2 手 EURGBP。
Given: leverage 1:500, equity USD 2,000, EURGBP bid 0.87230 and ask 0.87260. The client sells 2 lots short.
— 合约大小是多少？ / Contract size?
ans: 外汇的合约量是 100,000 单位/手 / FX contract size = 100,000 units per lot.

— 点值是多少？ / Pip value?
ans: Pip value = TickSize × Contract = 0.00001 × 100,000 = 1 GBP / pip

— 所需保证金是多少？ / Required margin?
ans: EURGBP is a cross pair, base currency = EUR, so we first compute EURUSD:
     EURUSD = EURGBP × GBPUSD = 0.87230 × 1.33437 = 1.16397
     Margin = Lots × Contract × EURUSD / Leverage = 2 × 100,000 × 1.16397 / 500 = 465.59 USD
     (Short EURGBP → use the bid = 0.87230.)

— 若止损比例为 50%，请问爆仓时 EURGBP 的价格是多少？（GBPUSD 取 1.33437）/ Liquidation price with 50 % stop-out? (Take GBPUSD = 1.33437.)
ans: Short → price must rise to liquidate.
     Condition: Equity = Margin × Stop-out Ratio.
     Per-lot pip value = 1 GBP / pip = 1.33437 USD / pip.
     2,000 − (P − 0.87230) × 2 × 100,000 × 1.33437 = 465.59 × 0.50
     → Stop-out price = 0.87892.

10)
已知：账户杠杆为 1:500，账户净值为 1,500 美元，EURJPY 报价：卖出价 162.276，买入价 162.298。USDJPY: 143.076。若客户买入 1 手 EURJPY，止损比例为 50%。
Given: leverage 1:500, equity USD 1,500, EURJPY bid 162.276 and ask 162.298, USDJPY 143.076. The client buys 1 lot long with stop-out ratio = 50 %.
— 所需保证金是多少？ / Required margin?
ans: EURJPY is a cross pair, base currency = EUR, so compute EURUSD first:
     EURUSD = EURJPY / USDJPY = 162.298 / 143.076 = 1.13435
     Margin = Lots × Contract × EURUSD / Leverage = 1 × 100,000 × 1.13435 / 500 = 226.87 USD

— 点值是多少？ / Pip value?
ans: Pip value = TickSize × Contract = 0.001 × 100,000 = 100 JPY / point
     Convert to USD: 100 / 143.076 = 0.70 USD / point.

— 爆仓时 EURJPY 的价格是多少？ / Liquidation price?
ans: Long → price must fall to liquidate.
     Condition: Equity = Margin × Stop-out Ratio.
     i.e. 1,500 − (162.298 − P) × 1 × 100,000 / 143.076 = 226.87 × 0.50
     → Stop-out price = 160.314.
